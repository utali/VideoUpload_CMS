Panels can also be tabbed. For this you'll need to use the tabset element and set panel-tabs="true". The other options are same as above.

<tabset panel-tabs="true" panel-class="{{demoTabbedPanelClass}}" heading="{{demoTabbedPanelHeading}}">
	<tab heading="Tab 1">
		<p>Lorem ipsum dolor sit amet consectetur adipisicing elit. Tenetur ipsam consectetur sint. Nobis facere illo iste quaerat sapiente doloribus deserunt et nam obcaecati recusandae possimus aperiam similique.</p>
	</tab>

	</tab>

Panels can optionally have controls through the panel-controls directive to provide extra options. In this example, the panel-control-collapse () control causes the panel to collapse and expand. You can create your own directives that augment panel behavior. Check it out!

<panel panel-class="{{demoPanelClass}}" heading="{{demoPanelHeading}}">
	<panel-controls>
        <a href="javascript:;"><i class="fa fa-cog"></i></a>
        <a href="javascript:;"><i class="fa fa-wrench"></i></a> 
        <a href=""><panel-control-collapse class="fa fa-chevron-down"></panel-control-collapse></a>
	</panel-controls>
