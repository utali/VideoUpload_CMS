Images

Add classes to an img element to easily style images in any project.

<img src="..." alt="..." class="img-rounded">

<img src="..." alt="..." class="img-circle">

<img src="..." alt="..." class="img-thumbnail">

Images in Forza, thanks to Bootstrap 3, can be made responsive-friendly via the addition of the .img-responsive class. This ensures that even large images scale down to the parent
